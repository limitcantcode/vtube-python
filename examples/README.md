# Examples

This directory contains example scripts demonstrating how to use vtube-python.

## Getting Started

1. Make sure you have VTube Studio running
2. Install vtube-python in development mode:
   ```bash
   pip install -e .
   ```
3. Run any example script:
   ```bash
   python examples/basic_connection.py
   ```

## Available Examples

Examples will be added as the library is developed.
